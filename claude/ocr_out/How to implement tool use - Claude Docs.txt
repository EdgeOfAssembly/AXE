Toolrunner (beta)
The tool runner provides an out-of-the-box solution for executing tools with Claude:
Instead of manually handling tool calls, tool results, and conversation management; the
tool runner
automatically:
Executes tools when Claude calls them
Handles the requestlresponse cycle
Manages conversation state
Provides type safety and validation
We recommend that yoU use the tool runner for most tool use implementations
The tool runner is currently in beta and available in the Python; TypeScript; and Ruby SDKs.
Automatic context management with compaction
The tool runner supports automatic compaction, which generates summaries when token
usage exceeds a threshold. This allows long-running agentic tasks to continue beyond
context window limits.
