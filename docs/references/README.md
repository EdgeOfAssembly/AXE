## Quick Command Lookup

### Essential Commands
```bash
# Start AXE
python axe.py

# Use specific agent
python axe.py --agent claude

# Multi-agent collaboration
python axe.py --collab

# Load session
python axe.py --load <session_name>
```

### In-Session Commands
```
/help           - Show available commands
/agent <name>   - Switch to different agent
/collab         - Enable collaborative mode
/workshop       - Access dynamic analysis tools
/save           - Save current session
/quit           - Exit AXE
```
