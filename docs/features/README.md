# AXE Features

This directory contains detailed documentation for AXE's core features and capabilities.

## Available Features

### [sandbox.md](sandbox.md)
**Sandboxed Command Execution with Bubblewrap**

Learn how to use AXE's security sandbox for isolated command execution:
- Bubblewrap integration
- Namespace isolation
- Filesystem restrictions
- Security policies
- Configuration options

### [write-blocks.md](write-blocks.md)
**File Manipulation and Write Blocks**

Comprehensive guide to file operations:
- READ blocks for reading files
- WRITE blocks for creating/modifying files
- APPEND operations
- Path handling
- Security considerations

### [parsers.md](parsers.md)
**Multi-Format Tool Call Parsing**

Understanding AXE's flexible tool call parser:
- XML function calls (Anthropic/OpenAI standard)
- Bash XML tags
- Shell code blocks
- AXE native blocks (READ/WRITE/EXEC)
- Simple XML tags
- Mixed format support
- Examples and best practices
