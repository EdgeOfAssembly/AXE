\doxysubsection{Detailed Description}
\begin{DoxyVerb}Compresses system prompts to reduce token usage.

Features:
- Remove verbose explanations
- Condense examples
- Abbreviate common instructions
- Preserve critical directives
\end{DoxyVerb}
