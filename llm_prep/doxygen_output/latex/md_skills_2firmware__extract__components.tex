Skill\+: firmware\+\_\+extract\+\_\+components Category\+: Firmware Extraction Recommended agents\+: llama (execution), claude (verification) Purpose\+: Automatically extract embedded filesystems, compressed sections, kernels and other components using binwalk

Key commands\+:
\begin{DoxyItemize}
\item binwalk -\/e $<$file$>$ → extract everything to \+\_\+$<$filename$>$.\+extracted/
\item binwalk --dd=\textquotesingle{}.\texorpdfstring{$\ast$}{*}\textquotesingle{} $<$file$>$ → extract all detected file types
\item binwalk -\/r -\/e $<$file$>$ → recursive extraction
\end{DoxyItemize}

\begin{DoxyItemize}

\item Look for interesting files\+: /etc/passwd, web UI files, scripts, certificates
\end{DoxyItemize}
