\hypertarget{md_README_autotoc_md866}{}\doxysection{\texorpdfstring{Overview}{Overview}}\label{md_README_autotoc_md866}
AXE (Agent e\+Xecution Engine) is a powerful terminal-\/based multiagent system designed for\+:


\begin{DoxyItemize}
\item {\bfseries{Coding Tasks}}\+: C, C++, Python development with multiple AI assistants
\item {\bfseries{Reverse Engineering}}\+: Analysis of DOS-\/era binaries, WAD files, executables
\item {\bfseries{Code Review}}\+: Security auditing and code quality analysis
\item {\bfseries{Multi-\/\+Provider Support}}\+: Use Claude, GPT, Llama, Grok, and Git\+Hub Copilot in one interface
\end{DoxyItemize}\hypertarget{md_README_autotoc_md867}{}\doxysubsection{\texorpdfstring{Why AXE?}{Why AXE?}}\label{md_README_autotoc_md867}

\begin{DoxyEnumerate}
\item {\bfseries{Unified Interface}}\+: Talk to multiple AI models through one terminal interface
\item {\bfseries{Tool Integration}}\+: Execute whitelisted commands (hexdump, objdump, gdb, etc.)
\item {\bfseries{Context-\/\+Aware}}\+: Automatically provides project context to agents
\item {\bfseries{Safe Execution}}\+: Command whitelist and directory access control
\item {\bfseries{Flexible Config}}\+: YAML/\+JSON configuration for easy customization
\end{DoxyEnumerate}
