\hypertarget{md_docs_2GITHUB__AGENT__INTEGRATION_autotoc_md299}{}\doxysection{\texorpdfstring{Review Flow}{Review Flow}}\label{md_docs_2GITHUB__AGENT__INTEGRATION_autotoc_md299}

\begin{DoxyEnumerate}
\item {\bfseries{Agent signals ready}}\+: Uses {\ttfamily \mbox{[}\mbox{[}GITHUB\+\_\+\+READY\+: ...\mbox{]}\mbox{]}} token
\item {\bfseries{Session pauses}}\+: Review interface appears
\item {\bfseries{Human reviews}}\+: Can view diff, approve, or reject
\item {\bfseries{If approved}}\+: Code is pushed to specified branch
\item {\bfseries{Optional PR creation}}\+: Human can create PR via {\ttfamily gh} CLI
\item {\bfseries{If rejected}}\+: Feedback is sent back to agents
\end{DoxyEnumerate}\hypertarget{md_docs_2GITHUB__AGENT__INTEGRATION_autotoc_md300}{}\doxysection{\texorpdfstring{Rejection and Feedback}{Rejection and Feedback}}\label{md_docs_2GITHUB__AGENT__INTEGRATION_autotoc_md300}
When rejecting a push request\+:


\begin{DoxyEnumerate}
\item Enter \textquotesingle{}r\textquotesingle{} at the review prompt
\item Provide feedback explaining why (e.\+g., "{}\+Need more tests"{})
\item Session resumes with feedback injected
\item Agents can address feedback and signal ready again
\end{DoxyEnumerate}\hypertarget{md_docs_2GITHUB__AGENT__INTEGRATION_autotoc_md301}{}\doxysection{\texorpdfstring{Requirements}{Requirements}}\label{md_docs_2GITHUB__AGENT__INTEGRATION_autotoc_md301}

\begin{DoxyItemize}
\item {\bfseries{No git repo}}\+: Warning displayed, feature disabled
\item {\bfseries{SSH not configured}}\+: Push fails with clear error
\item {\bfseries{No changes}}\+: Commit fails with "{}nothing to commit"{} message

\item Never bypasses human review
\item No automatic PR merging
