The Workshop module provides dynamic analysis tools for AXE, themed after woodworking implements. These tools enable advanced code analysis, vulnerability detection, and runtime monitoring.\hypertarget{md_workshop__quick__reference_autotoc_md1401}{}\doxysection{\texorpdfstring{Tools}{Tools}}\label{md_workshop__quick__reference_autotoc_md1401}
