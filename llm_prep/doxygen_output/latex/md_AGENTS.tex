{\bfseries{AXE (Agent e\+Xecution Engine)}} is a terminal-\/based multiagent coding assistant designed for C, C++, Python, and reverse engineering tasks. It orchestrates multiple specialized AI agents (Claude, GPT, Llama, Grok, etc.) to collaborate on complex coding, debugging, security auditing, and binary analysis tasks.\hypertarget{md_AGENTS_autotoc_md42}{}\doxysection{\texorpdfstring{Development Environment}{Development Environment}}\label{md_AGENTS_autotoc_md42}

\hypertarget{md_AGENTS_autotoc_md57}{}\doxysection{\texorpdfstring{Build Guidelines}{Build Guidelines}}\label{md_AGENTS_autotoc_md57}
AXE emphasizes {\bfseries{silent builds}} to reduce token consumption and improve log readability.\hypertarget{md_AGENTS_autotoc_md58}{}\doxysubsection{\texorpdfstring{Default Build Approach}{Default Build Approach}}\label{md_AGENTS_autotoc_md58}
Use silent/quiet flags for all build systems\+:
\begin{DoxyItemize}
\item {\bfseries{Make}}\+: {\ttfamily make -\/s V=0} or {\ttfamily make -\/-\/silent}
\item {\bfseries{CMake}}\+: {\ttfamily cmake -\/-\/build . -\/-\/ VERBOSE=0}
\item {\bfseries{Ninja}}\+: Silent by default; use {\ttfamily -\/v} only for debugging
\item {\bfseries{Gradle}}\+: {\ttfamily gradle -\/-\/quiet} or {\ttfamily gradle -\/q}
\item {\bfseries{Maven}}\+: {\ttfamily mvn -\/q} or {\ttfamily mvn -\/-\/quiet}
\end{DoxyItemize}\hypertarget{md_AGENTS_autotoc_md59}{}\doxysubsection{\texorpdfstring{When to Use Verbose Output}{When to Use Verbose Output}}\label{md_AGENTS_autotoc_md59}
